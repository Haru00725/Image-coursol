## ✨ Key Features

* **♾️ Seamless Infinite Scrolling:** A true infinite loop with a dual-index system ensuring zero visual jumps.
* **▶️ Auto-Play & Precision Control:** Automatic progression with play/pause functionality, timed perfectly with 1.5s ease-in-out-cubic transitions.
* **🖼️ Dynamic Custom Uploads:** Replace default assets on the fly.
* **✨ Glassmorphism UI:** Sophisticated styling featuring a 27.15px backdrop blur.
